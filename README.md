# 🧠 ThinkTutor – AI Powered Teaching Assistant

An intelligent AI-powered learning assistant designed to help students study smarter, prepare for exams, solve doubts, and understand concepts interactively using Google Gemini AI.

---

# 🚀 Overview

ThinkTutor is an educational AI platform that acts like a personal teaching assistant for students.  
It provides intelligent responses, concept explanations, academic support, and personalized learning assistance through a modern responsive interface.

The project combines AI, web technologies, and interactive UI design to create a smart learning ecosystem.

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ThinkTutor.git
cd ThinkTutor
```

---

## 2️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Setup Environment Variables

Create a `.env` file in the root directory.

Example:

```env
GEMINI_API_KEY=your_api_key_here
```

---

# ▶️ Run the Project

## Flask Version

```bash
python app.py
```
